In Operator

  • In opeator adalah operator yang bisa dilakukan untuk mengecek apakah sebuah property ada di dalam object atau tidak
  • jika property tersebut ada di oject, maka hasilnya true, sedangkan jika tidak maka hasilnya false
  • tidak hanya di object, In juga bisa dilakukan untuk mengecek index di array

    • Peringatan

  • In operator hanya akan mengecek apakah property atau index ada atau tidak
  • jadi walaupun nilai property atau index nya undefined atau null, maka tetap akan di anggap ada
